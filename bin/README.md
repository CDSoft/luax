# Prebuilt LuaX interpreter

This directory contains prebuilt LuaX implementations (in pure Lua):

- `luax.lua`: standalone LuaX interpreter for standard Lua interpreter
- `luax-pandoc.lua`: standalone LuaX interpreter for the Pandoc Lua interpreter
- `bang.lua`: Bang embedding the LuaX runtime for standard Lua interpreters
- `bang.luax`: Bang with no LuaX runtime, meant for LuaX interpreters
- `ypp.lua`: ypp embedding the LuaX runtime for standard Lua interpreters
- `ypp-pandoc.lua`: ypp embedding the LuaX runtime for the Pandoc Lua interpreter
- `ypp.luax`: ypp with no LuaX runtime, meant for LuaX interpreters
- `lsvg.lua`: SVG image generator
