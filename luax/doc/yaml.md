# yaml: YAML Module for Lua

```lua
local yaml = require "yaml"
```

The yaml package is taken from
[lua-tinyyaml](https://github.com/zepinglee/lua-tinyyaml).

The lua-tinyyaml package is a lightweight YAML parser written in pure Lua.
It supports a subset of the YAML 1.2 specifications.

Please check [lua-tinyyaml](https://github.com/zepinglee/lua-tinyyaml) for further information.
